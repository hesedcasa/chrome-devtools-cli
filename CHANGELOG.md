# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.2] - 2025-11-14

### Added
- Release automation script for GitHub and NPM publishing

## [1.1.0] - 2025-01-13

### Changed
- Improved version retrieval
- Refactored entire project to modular structure

## [1.0.0] - Initial Release

### Added
- CLI tool to connect and interact with Chrome DevTools MCP server
- Interactive REPL interface
- 26 browser control commands
- Headless command execution mode
- TypeScript support with full type definitions
